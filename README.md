# Engrenarium

O Engrenarium é uma ferramenta web interativa para modelagem e visualização de sistemas de engrenagens planetárias, com foco em ensino e análise cinemática de transmissões.

## Acesso
- Aplicação: https://barbudenge.github.io/engrenarium/
- Repositório: https://github.com/Barbudenge/engrenarium

## Funcionalidades (visão geral)
- Modelagem e simulação de engrenagens planetárias
- Suporte a múltiplos estágios (conforme implementação do projeto)
- Visualização interativa para estudo e demonstrações
- Exportação/registro de resultados (conforme recursos disponíveis)

## Tecnologias
- TypeScript
- Vite
- Renderização/visualização 3D (conforme stack do projeto)
- GitHub Pages com publicação automatizada via GitHub Actions

## Contexto acadêmico
Projeto desenvolvido no âmbito do LASME — UFSJ, com finalidade didática e de pesquisa.
